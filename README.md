# 🔍 Python Job Scraper – TimesJobs

---

## 📊 Project Overview
This project is a Python-based web scraping tool that extracts job listings from TimesJobs. It allows users to filter out jobs based on unwanted skills, helping them focus only on relevant opportunities.

---

## 🎯 Objectives
- Scrape real-time job data from TimesJobs  
- Filter jobs based on user-defined skills  
- Display relevant job listings with key details  
- Automate job search with periodic updates  

---

## ⚙️ Features
- Live job scraping using HTTP requests  
- Skill-based filtering system  
- Displays company name, required skills, and job link  
- Auto-refresh every 10 minutes  
- Simple and lightweight script  

---

## 🛠️ Technologies Used
- Python  
- BeautifulSoup (Web Scraping)  
- Requests (HTTP Handling)  
- Time Module (Automation)  

---

## 📈 Key Functionality
- Extracts job postings from HTML structure  
- Filters out jobs containing unwanted skills  
- Provides direct links to job listings  
- Continuously updates results at fixed intervals  

---

## 🚀 Project Outcome
Built an automated job scraping tool that simplifies job searching by filtering out irrelevant roles and providing updated listings in real time.
